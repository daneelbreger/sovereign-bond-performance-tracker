# Sovereign Bond Performance Tracker

A simple Python script that loads sovereign yield data from a CSV file and plots the yield trend over time.

### Features
- Beginner Python code
- Uses pandas and matplotlib
- Works with any CSV containing Date/Yield columns

### How to run
1. Install dependencies:
   pip install pandas matplotlib

2. Run the script:
   python sovereign_bond_tracker.py

### Files
- `sovereign_bond_tracker.py` — main script
- `sovereign_yields.csv` — sample data
